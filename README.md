# Projeto em PHP que reúne tecnologias Web
Projeto da matéria "Linguagem de Programação IV - Internet" para P2. 1º Semestre 2026.

Elaborar um projeto com temática apresentada na Figura 1 - Seção II, com o propósito de demonstrar o uso de diversos recursos (elecandos d Seção III). O projeto pode conter mais de uma tela com recursos de HTML, CSS e JavaScript. Porém, deve conter todos os itens apresentados na Seção III com foco em PHP.
